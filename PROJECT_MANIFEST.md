## 🔄 Usage Workflow

1. **Import Components**
   ```python
   from jarvis_os import JarvisOS, AgentConfig
   ```

2. **Configure Agent**
   ```python
   config = AgentConfig(name="agent1")
   ```

3. **Create Agent**
   ```python
   agent = JarvisOS(config)
   ```

4. **Execute Tasks**
   ```python
   result = await agent.execute_task(...)
   ```

5. **Monitor & Optimize**
   ```python
   metrics = agent.get_metrics()
   ```
